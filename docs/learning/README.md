# Learning by Commit Guide (LCG) の学習資産

このディレクトリは、日々のコミット差分（Diff）から得られた学びを蓄積し、チーム全体の知見へ変換するための資産保管場所です。

## 目的
* **Guardrail teaches humans:** 仕組みのガードが学習を生むという OS の原則に基づき、CI/Guardrail の指摘や深いコードレビューの議論を教材化する。
* **小さな資産化:** 同じミスや設計の落とし穴をチームで共有し、継続的にチームの筋肉（開発力・レビュー力）を増やす。

## 運用方針
1. 有意義な差分（Diff）、特に設計的気付きやアンチパターンの回避につながる過去のコード変更があった場合、Markdown 形式で資産として抽出します。
2. 資産化されたファイルは `docs/learning/commits/<commit-hash>.md` 等の形で保管されます。
3. 今後の Pull Request や新機能開発時、類似の課題に直面した場合にエージェントや開発者が素早く参照できる知識ベースとして機能します。
